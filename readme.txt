1) Параметры для запуска TrainingSampleMaker.exe:
	(1) Путь к изображению
	(2) -fg -- Путь к папке, куда будут сохраняться выборка с нейронами (папка должна существовать)
	(3) -bg -- Путь к папке, куда будут сохраняться выборка с фоном (папка должна существовать)
  В папки сохраняется xml файл с описанием путей и сами изображения выборки
  
2) В bin/run.bat - пример запуска для нескольких изображений. Нужно заменить Tv104.jpg и Tv244.jpg на свои изображения и на каждом изображении последовательно сделать разметку.

3) В приложении opencv отсутствует UI, поэтому все взаимодействие только на горячих клавишах и мышкой:
	(1) Горячая клавиша h показывает/убирает помощь
	(2) Клавиши w, a, s, d для навигации по изображению; e, q для zoom
	(3) Клавиши 1, 2, 3, 4 - подбор подходящего размера окна для нейронов (1 - 24 пикселя, 2 - 36 пикселя, 3 - 48 пикселя, 4 - 60 пикселя)
	(4) После разметки можно нажать на Ctrl+f - будет сгенерирована выборка фона
	(5) Ctrl+s - делает сохранение выборки, выборка фона будет сгенерирована автоматически
	(6) Enter - закрыть программу и сделать сохранение
	(7) Esc - закрыть программу без сохранения
	(8) Ctrl+R - удаляет всю разметку
	(9) Правая кнопка мыши используется для навигации по изображению
	(10) Левая кнопка мыши добавляет/удаляет размеченную область. Если под курсом нет размеченной области - она добавляется, если есть - удаляется
	(11) Ctrl+z - Удаляет последнюю разметку
	
4) В Windows, если разметка для изображения уже сделана - то она подгружается и ее можно редактировать. 
	Чтобы она работала в маке и других системах нужно дописать функцию получения файлов из папки, в коде эта функция находится дефайном WIN32

5) clean.bat - удаляет всю разметку для всех изображений

6) Проект написан на MS Visual Studio 2010 с билиотекой opencv 2.4.9. Скомпилированный для x32 системе есть в папке bin
